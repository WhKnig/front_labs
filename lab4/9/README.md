# Лабораторная работа 4: TypeScript

Задание из файла `9.pdf`.

## Структура проекта

- `src/user.ts` — Задания 2-3.
- `src/distance.ts` — Задание 4.
- `src/tree.ts` — Задание 5.
- `src/patterns.ts` — Задание 6.
- `src/index.ts` — Главный файл для запуска.

## Запуск

1. Установить: npm install
2. Скомпилировать: npm run build
3. Запустить: npm start
4. Отслеживание изменений: npm run watch
