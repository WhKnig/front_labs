# Лабораторная работа 4: SCSS/SASS/Less
Задание из файла `8.pdf`.

## Структура проекта
- `index.html` — главная страница с заданиями 2 и 3.
- `numbers.scss` / `layout.scss` — основная реализация на **SCSS**.
- `numbers.sass` / `layout.sass` — реализация на **SASS**.
- `numbers.less` / `layout.less` — реализация на **Less**.


### SASS (SCSS)
npm run sass:watch

npm run sass:build

### Less
npm run less:build
